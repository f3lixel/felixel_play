Lege die Emulatoren hier lokal ab:

- `dolphin/Dolphin-x64/Dolphin.exe` fuer Wii und Wii U
- `sudachiemulator.org-winpc-1-0-15/sudachi.exe` fuer Switch
- `ryujinx-canary-1.3.287-win_x64/publish/Ryujinx.exe` als Switch-Fallback

Die App startet diese lokalen Emulatoren automatisch, wenn du auf ein Spiel klickst.
Die Emulator-Dateien selbst werden nicht in Git aufgenommen.
