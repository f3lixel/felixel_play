Lege deine eigenen, legal gesicherten ROM-Dateien hier ab:

- `wii/` fuer Wii-Dateien: `.iso`, `.wbfs`, `.rvz`, `.gcz`, `.ciso`
- `wiiu/` fuer Wii-U-Dateien: `.wud`, `.wux`, `.rpx`, `.iso`
- `switch/` fuer Switch-Dateien: `.nsp`, `.xci`, `.nca`, `.nro`

Die App scannt diese Ordner beim Start automatisch.
